# Security Policy

## Supported Versions
Only the latest release of `insta-repo` is supported with security updates. 

## Reporting a Vulnerability
If you discover a bypass or a vulnerability in the heuristic engine, please do not open a public issue. Instead, reach out directly to the maintainer via the contact information on the `credkellar-boop` GitHub profile. 

We will attempt to acknowledge and resolve the issue as quickly as possible.
